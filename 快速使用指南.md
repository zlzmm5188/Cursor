## 🚀 快速启动

### 方法1: 直接打开（最简单）

```bash
# 在浏览器中直接打开
open admin-panel/index.html
```

### 方法2: 使用HTTP服务器（推荐）

```bash
# 进入目录
cd admin-panel

# 使用Python启动（如果已安装）
python3 -m http.server 8083

# 或使用Node.js（如果已安装）
npx http-server -p 8083
```

然后访问: `http://localhost:8083/index.html`

## 🔌 API接口

用户管理使用的API：

| 功能     | API端点                        | 方法 |
| -------- | ------------------------------ | ---- |
| 用户列表 | `/api/user/list`               | GET  |
| 创建用户 | `/api/user/create`             | POST |
| 更新用户 | `/api/user/update/{id}`        | POST |
| 冻结用户 | `/api/user/freeze`             | POST |
| 解冻用户 | `/api/user/unfreeze`           | POST |
| 团队树   | `/api/team/tree/{userId}`      | GET  |
| 登录日志 | `/api/user/login-log/{userId}` | GET  |
